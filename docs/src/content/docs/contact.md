Five routes. Pick the one that fits.

## General questions, ideas, partnerships

Email **[`hello@sealstack.ai`](mailto:hello@sealstack.ai)**.

This is the inbox for press, partnerships, conference talk requests,
consulting inquiries, and "does SealStack do X?" questions where a
GitHub issue isn't the right venue.

Expected response time: a few business days. It's one person answering.

## Bugs and feature requests

Open an issue on GitHub: **[github.com/bwiemz/sealstack/issues](https://github.com/bwiemz/sealstack/issues)**.

Please search before filing — duplicates happen. Include the
SealStack version (`sealstack --version`), your OS, and a minimal
reproduction if you can.

## Security vulnerabilities

**Do not open a public issue** for security problems.

Email **[`security@sealstack.ai`](mailto:security@sealstack.ai)** instead,
following the disclosure policy in
[`SECURITY.md`](https://github.com/bwiemz/sealstack/blob/main/SECURITY.md).

Expect an acknowledgement within 72 hours. Coordinated disclosure is the
default — we'll work with you on a timeline before anything is public.

## Code of Conduct reports

Violations of the [Code of Conduct](https://github.com/bwiemz/sealstack/blob/main/CODE_OF_CONDUCT.md)
go to **[`conduct@sealstack.ai`](mailto:conduct@sealstack.ai)**.

Reports are handled confidentially by the project owner.

## Trademark inquiries

Permission to use the SealStack name or logo, or reports of trademark
misuse, go to **[`trademark@sealstack.ai`](mailto:trademark@sealstack.ai)**.

See the full [`TRADEMARK-POLICY.md`](https://github.com/bwiemz/sealstack/blob/main/TRADEMARK-POLICY.md)
for what permitted vs. prohibited use looks like.
