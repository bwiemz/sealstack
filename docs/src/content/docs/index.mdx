## What it is

ContextForge is an open-source, enterprise-grade context platform. It ingests
data from your tools, indexes it with hybrid BM25+vector retrieval, and exposes
MCP tools plus a REST API that every AI agent in your organization can call.

Every search returns a **receipt** — the sources it read, the policies it
enforced, and the time each stage cost. Those receipts are first-class
primitives, not an afterthought.
